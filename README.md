# MarkView Lab

Static local clone inspired by the MarkLook landing-preview page.

## Run

From this directory:

```bash
python3 -m http.server 4173
```

Then open:

`http://localhost:4173`

## Included

- Curated Markdown preview themes
- Language switcher
- Light / dark / system appearance
- Font size stepper
- Markdown demo content
- Drag-and-drop local `.md` preview
- Paste-to-preview
- Code highlighting, KaTeX, Mermaid
